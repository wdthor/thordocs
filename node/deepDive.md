# Node Deep Dive

## V8 Engine

- V8 is a JavaScript engine that is used by Node.
- Node and V8 are written in C++
- V8 converts JavaScript code into machine code that can be executed by a microprocessor.
- V8 adds a number of features to JavaScript that are not part of the ECMAScript specification.

### Conceptual Aside

#### Microprocessor

- A microprocessor is a small computer that can perform basic operations like addition, subtraction, and comparison.
- We give it instructions to perform tasks.

#### JavaScript Engine and the ECMAScript Specification

- A JavaScript engine is a program that can convert JavaScript code into something the computer processor can understand.
- The ECMAScript specification is a set of rules that define how JavaScript should work.
