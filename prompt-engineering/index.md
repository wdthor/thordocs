# Prompt Engineering

## What is Prompt Engineering ?

- Prompt : instructions and context provided to an AI for a certain task
- Prompt Engineering : the practice of developing and optimizing prompts to efficiently use an AI for a certain task
- Summary : A multidisciplinary branch of engineering focused on interacting with Artificial Intelligence through the integration of fields including Software Engineering, Machine Learning, Cognitive Science, Business, Phylosophy, Psychiatry, and Computer Science

## What is Applied Prompt Engineering ?

- Applying prompt engineering principles, knowledge, and skills to real-world tasks

Note: Just asking a question, is not prompt engineering

## Multi-modality

GPT and and advanced LLM can :

- Accept and generate text
- Accept images
- Generate images
- Browse internet
- Execute code
