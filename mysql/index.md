# Introduction

## What is a Database ?

- A collection of data
- A method for accessing and manipulating that data
- A structured set of computerized data with an accessible interface

## DBMS - DataBase Management System

- The interface between the app and the database
- Ex : PostgreSQL, MySQL, Oracle Database, SQLite

## SQL - Structured Query Language

- The language we use to "talk" to our databases
- Ex:
  - Find all users
  - Find all users who are 18 years old
  - Add a new user with username "Jumbojim",
  - Delete every single user
- SQL is a standard that DBMS use
