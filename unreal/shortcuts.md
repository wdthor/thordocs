# Shortcuts

## Unreal interface shortcuts

| Shortcuts         |           Description           |
| ----------------- | :-----------------------------: |
| shift + p         | Enter Play In Editor mode (PIE) |
| q                 |          Select object          |
| w                 |           Move object           |
| e                 |          Rotate object          |
| r                 |          Scale object           |
| f                 |         Focus on object         |
| alt + left click  |  Rotate view around an object   |
| alt + right click |      Zoom in/out an object      |

## Move Tool

| Shortcuts           |                                      Description                                       |
| ------------------- | :------------------------------------------------------------------------------------: |
| shift + move object |                             Move the view with the object                              |
| alt + move object   | **Duplicate** and move the object (The same can be applied with the **Rotation** tool) |

## Unreal PIE shortcuts

| Shortcuts |                  Description                   |
| --------- | :--------------------------------------------: |
| alt + F1  |           Display mouse cursor back            |
| ESC       |                 Exit PIE mode                  |
| F8        | Detach from player (pawn - only exists in PIE) |
